# Product Feedback Analysis – Myntra

## Overview
This project analyzes 30 Myntra customer reviews using AI-assisted research techniques.

## Objectives
- Identify customer pain points
- Categorize feedback into themes
- Recommend product improvements
- Prioritize features

## Key Findings
1. Customer Support Issues
2. Delivery Reliability Problems
3. Return & Exchange Friction

## Recommended Features
- OTP + Photo Delivery Verification
- Evidence-First Support Escalation
- Improved Return Workflow

## Skills Demonstrated
- Product Management
- User Research
- AI Tools
- Data Analysis
- Feature Prioritization
